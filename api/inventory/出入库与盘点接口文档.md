> 更新日期：2026-05-05

## 一、入库单接口

### 1.1 创建入库单

- **接口路径**：`POST /inventory/inbound/create`
- **权限标识**：`inventory:inbound:create`

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| inboundType | String | 是 | 入库类型：PURCHASE/PRODUCTION/RETURN_SALES/TRANSFER |
| warehouseId | Long | 是 | 目标仓库ID |
| sourceType | String | 否 | 来源单据类型 |
| sourceId | Long | 否 | 来源单据ID |
| sourceNo | String | 否 | 来源单据编号 |
| remark | String | 否 | 备注 |
| lines | List | 是 | 入库行列表 |
| lines[].inventoryType | String | 是 | 库存类型：SKU/MATERIAL |
| lines[].skuId | Long | 成品必填 | SKU ID |
| lines[].materialId | Long | 原料必填 | 物料ID |
| lines[].batchNo | String | 否 | 批次号 |
| lines[].plannedQty | BigDecimal | 是 | 计划入库数量 |
| lines[].actualQty | BigDecimal | 是 | 实际入库数量 |
| lines[].locationId | Long | 否 | 入库库位ID |
| lines[].unitCost | BigDecimal | 否 | 单位成本 |

### 1.2 确认入库

- **接口路径**：`POST /inventory/inbound/confirm`
- **权限标识**：`inventory:inbound:confirm`
- **说明**：确认入库后自动触发库存变更（changeInventory），入库单状态变为 CONFIRMED

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| inboundId | Long | 是 | 入库单ID |

**业务规则：**
- 只有 DRAFT 状态允许确认
- PURCHASE/RETURN_SALES → 质检库存增加
- PRODUCTION/TRANSFER → 可用库存增加
- 确认后自动写操作流水

### 1.3 查询入库单详情

- **接口路径**：`POST /inventory/inbound/detail`

### 1.4 分页查询入库单

- **接口路径**：`POST /inventory/inbound/page`

## 二、出库单接口

### 2.1 创建出库单

- **接口路径**：`POST /inventory/outbound/create`
- **权限标识**：`inventory:outbound:create`

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| outboundType | String | 是 | 出库类型：SALES/MATERIAL/TRANSFER/RETURN_PURCHASE |
| warehouseId | Long | 是 | 源仓库ID |
| lines | List | 是 | 出库行列表 |
| lines[].inventoryType | String | 是 | 库存类型：SKU/MATERIAL |
| lines[].skuId | Long | 成品必填 | SKU ID |
| lines[].plannedQty | BigDecimal | 是 | 计划出库数量 |
| lines[].actualQty | BigDecimal | 否 | 实际出库数量 |
| lines[].allocationId | Long | 销售出库 | 关联的预留记录ID |

### 2.2 确认出库（发货确认）

- **接口路径**：`POST /inventory/outbound/confirm`
- **权限标识**：`inventory:outbound:confirm`
- **说明**：发货确认后自动扣减库存，出库单状态变为 SHIPPED

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| outboundId | Long | 是 | 出库单ID |

**业务规则：**
- 只有 DRAFT/CONFIRMED/PICKING 状态允许发货确认
- SALES → 从锁定库存扣减
- MATERIAL/TRANSFER → 从可用库存扣减
- RETURN_PURCHASE → 从质检库存扣减
- 库存不足时拒绝操作

### 2.3 查询出库单详情

- **接口路径**：`POST /inventory/outbound/detail`

### 2.4 分页查询出库单

- **接口路径**：`POST /inventory/outbound/page`

## 三、盘点单接口

### 3.1 创建盘点单

- **接口路径**：`POST /inventory/stocktaking/create`
- **权限标识**：`inventory:stocktaking:create`

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| stocktakingType | String | 是 | 盘点类型：FULL/CYCLE/SAMPLE/DYNAMIC |
| countMode | String | 是 | 盘点模式：OPEN=明盘/BLIND=盲盘 |
| warehouseId | Long | 是 | 盘点仓库ID |
| zoneCode | String | 否 | 盘点库区 |
| plannedDate | String | 否 | 计划盘点日期 |

### 3.2 开始盘点

- **接口路径**：`POST /inventory/stocktaking/start`
- **说明**：冻结被盘库位，状态变为 IN_PROGRESS

### 3.3 录入实盘数量

- **接口路径**：`POST /inventory/stocktaking/record-count`

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| stocktakingId | Long | 是 | 盘点单ID |
| lineId | Long | 是 | 盘点行ID |
| actualQty | BigDecimal | 是 | 实盘数量 |

### 3.4 提交盘点结果

- **接口路径**：`POST /inventory/stocktaking/submit`
- **说明**：自动计算差异，标记需复盘的行（差异率>5%），状态变为 DIFF_REVIEW

### 3.5 审核差异并调整库存

- **接口路径**：`POST /inventory/stocktaking/review`
- **权限标识**：`inventory:stocktaking:review`
- **说明**：盘盈调增、盘亏调减，完成后解冻库位，状态变为 COMPLETED

### 3.6 查询盘点单详情

- **接口路径**：`POST /inventory/stocktaking/detail`
- **说明**：盲盘模式下，盘点进行中时 systemQty 字段不返回

### 3.7 分页查询盘点单

- **接口路径**：`POST /inventory/stocktaking/page`

## 四、入库类型与库存操作映射

| 入库类型 | 操作类型 | 目标库存 |
|---------|---------|---------|
| PURCHASE | INBOUND_PURCHASE | qc_qty += N |
| PRODUCTION | INBOUND_PRODUCTION | available_qty += N |
| RETURN_SALES | INBOUND_RETURN | qc_qty += N |
| TRANSFER | INBOUND_PRODUCTION | available_qty += N |

## 五、出库类型与库存操作映射

| 出库类型 | 操作类型 | 来源库存 |
|---------|---------|---------|
| SALES | OUTBOUND_SALES | locked_qty -= N |
| MATERIAL | OUTBOUND_MATERIAL | available_qty -= N |
| TRANSFER | OUTBOUND_MATERIAL | available_qty -= N |
| RETURN_PURCHASE | QC_FAIL | qc_qty -= N |

## 六、错误码

| 错误码 | 含义 |
|--------|------|
| 50009 | 入库单状态不允许此操作 |
| 50010 | 出库单状态不允许此操作 |
| 50011 | 盘点单状态不允许此操作 |
| 50012 | 该库位已冻结，禁止出入库操作 |
