# 全局错误码与通用结构说明

## 统一响应格式

所有接口统一返回 `R<T>` 格式：

| 字段 | 类型 | 说明 |
|------|------|------|
| code | int | 0=成功，非零为错误码 |
| message | String | 提示信息 |
| data | T | 业务数据 |

## 分页响应结构

分页查询接口返回 `R<IPage<T>>`，`data` 结构如下：

| 字段 | 类型 | 说明 |
|------|------|------|
| records | List\<T\> | 当前页数据列表 |
| total | long | 总记录数 |
| size | long | 每页条数 |
| current | long | 当前页码 |
| pages | long | 总页数 |
