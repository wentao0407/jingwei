# 成本核算接口文档

> 模块：cost（成本核算）
> 版本：v1.0
> 日期：2026-05-05
> 前置依赖：inventory 模块（库存变更）、order 模块（生产订单）

---

## 一、概述

成本核算采用**移动加权平均法**，覆盖从面料采购到成品出库的完整成本流转链路：

```
采购入库 → 面料单位成本 = 采购单价（含税）
    ↓
领料出库 → 领料成本 = 面料单位成本 × 领料数量
    ↓
生产入库 → 成品单位成本 = Σ(领料成本) / 完工数量
    ↓
销售出库 → 销售成本 = 成品单位成本 × 出库数量
```

成本记录由系统自动触发，无需人工干预：
- **领料出库时**：自动记录领料成本（issue_qty × 领料时 unit_cost）
- **生产入库时**：自动归集总领料成本，计算成品单位成本
- **采购入库时**：自动计算加权平均成本

## 二、成本查询

### 2.1 查询生产订单成本详情

**接口路径：** `POST /cost/detail`
**权限标识：** `cost:query:detail`

**请求参数：**

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| productionOrderId | Long | 是 | 生产订单ID |
| productionLineId | Long | 是 | 生产订单行ID（颜色款） |

**响应结构：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1,
    "productionOrderId": 100,
    "productionLineId": 10,
    "materialCost": 90031.50,
    "trimCost": 38250.00,
    "packagingCost": 6300.00,
    "totalCost": 134581.50,
    "completedQty": 900,
    "unitCost": 149.54,
    "updatedAt": "2026-05-05T10:30:00"
  }
}
```

**业务规则：**
- 返回指定生产订单行的成本归集数据
- 包含面料、辅料、包材三项分类成本
- unitCost = totalCost / completedQty

## 三、成本计算规则

### 3.1 领料成本

```
领料成本 = 领料数量 × 领料时物料单位成本
```

- 领料时物料单位成本 = 该物料库存的当前加权平均成本
- 成本金额精确到分（2位小数，四舍五入）

### 3.2 成品单位成本

```
成品单位成本 = 生产订单总领料成本 / 完工数量
```

- 总领料成本 = 面料成本 + 辅料成本 + 包材成本
- 完工数量 = 生产入库的实际数量
- 单位成本精确到分

### 3.3 移动加权平均

```
新单位成本 = (原库存金额 + 本次入库金额) / (原库存数量 + 本次入库数量)
```

**示例：**
- 原库存：100件 × 50元 = 5,000元
- 入库：200件 × 55元 = 11,000元
- 新单位成本 = (5,000 + 11,000) / (100 + 200) = 53.33元/件

## 四、成本与库存的联动

| 库存操作 | 成本影响 |
|---------|---------|
| INBOUND_PURCHASE（采购入库） | 更新物料加权平均成本 |
| OUTBOUND_MATERIAL（领料出库） | 记录领料成本到 cost_material_issue |
| INBOUND_PRODUCTION（生产入库） | 计算成品加权平均成本 |
| OUTBOUND_SALES（销售出库） | 销售成本 = 成品单位成本 × 出库数量 |
