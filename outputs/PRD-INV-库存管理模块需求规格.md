# PRD-INV 库存管理模块需求规格

> 模块编号：INV  
> 优先级：P0（第二期交付）  
> 前置依赖：基础数据模块（SKU、物料、仓库/库位）、采购管理（到货检验）、订单管理（库存预留）

---

## 一、模块概述

库存管理模块是经纬系统的数据交汇点，管理成品和原料的四类库存（可用/锁定/质检/在途），确保库存数据准确、可追溯、不超卖。

---

## 二、用户故事

### US-INV-001 成品库存查询

**作为** 业务员/计划员/仓库主管  
**我希望** 按SKU查看各仓库的库存情况  
**以便** 了解成品可卖数量和存放位置

**验收标准：**

1. 支持按款式/颜色/尺码/仓库/季节筛选
2. 展示：可用库存、锁定库存、质检库存、实际库存、在途数量
3. 可展开查看各批次的库存明细（批次号、数量、库位、入库日期）
4. 支持矩阵视图：行为颜色，列为尺码，单元格为可用库存
5. 查询结果支持导出Excel

---

### US-INV-002 原料库存查询

**作为** 采购员/计划员/仓库主管  
**我希望** 按物料查看原料库存情况  
**以便** 了解面料/辅料的可用数量和批次信息

**验收标准：**

1. 支持按物料类型/物料名称/仓库/批次筛选
2. 展示：可用数量、锁定数量、质检数量、实际库存、在途数量、卷数（面料）
3. 可展开查看各批次明细（批次号、供应商、采购订单号、入库日期）
4. 面料库存额外展示：卷数、单卷平均米数

---

### US-INV-003 库存预留与释放

**作为** 系统（自动）  
**我希望** 销售订单确认时自动预留可用库存，取消时自动释放  
**以便** 防止库存被重复分配导致超卖

**验收标准：**

1. 销售订单确认 → 自动按SKU逐个预留可用库存
2. 库存充足 → 全额预留，锁定库存增加，可用库存减少
3. 库存不足 → 预留可用部分，记录缺口
4. 销售订单取消 → 释放预留，锁定库存减少，可用库存增加
5. 预留记录可查看：关联订单、预留数量、已出库数量、剩余预留
6. 预留超过7天未出库 → 自动释放（可配置天数）

---

### US-INV-004 入库单管理

**作为** 仓库主管  
**我希望** 管理各类入库单据  
**以便** 记录货物进入仓库的过程

**验收标准：**

1. 入库类型：采购入库、生产入库、退货入库、调拨入库
2. 入库单关联业务来源（采购订单/生产订单/销售订单退货）
3. 入库单行指定：SKU或物料、计划数量、实际数量、库位、批次号
4. 采购入库 → 质检库存增加
5. 生产入库 → 可用库存增加
6. 入库确认后自动写库存操作流水
7. 每次入库操作记录before/after库存数量

---

### US-INV-005 出库单管理

**作为** 仓库主管  
**我希望** 管理各类出库单据  
**以便** 记录货物离开仓库的过程

**验收标准：**

1. 出库类型：销售出库、领料出库、调拨出库、退货出库
2. 出库单关联业务来源和预留记录
3. 出库单行指定：SKU或物料、数量、批次、库位
4. 销售出库从锁定库存扣减
5. 领料出库从可用库存扣减
6. 出库时可用库存不足 → 拒绝操作，提示"库存不足"
7. 发货确认后才扣减库存（不是出库单创建时）

---

### US-INV-006 调拨管理

**作为** 仓库主管  
**我希望** 管理跨仓库的库存调拨  
**以便** 在仓库间调配货物

**验收标准：**

1. 创建调拨单指定源仓库、目标仓库、调拨物料/SKU及数量
2. 调拨确认 → 源仓可用库存减少
3. 调拨在途 → 货物在运输中，两仓都不算
4. 调拨完成 → 目标仓可用库存增加
5. 调拨在途状态可查看
6. 调拨不改变批次号

---

### US-INV-007 库存盘点

**作为** 仓库主管/仓管员  
**我希望** 定期盘点库存  
**以便** 发现和纠正系统库存与实际库存的差异

**验收标准：**

1. 支持全面盘点、循环盘点、抽盘三种类型
2. 创建盘点单时选择仓库、库区、盘点类型、明盘或盲盘
3. 系统自动生成待盘SKU/物料列表
4. 盲盘模式：盘点页面不展示系统数量
5. 盘点人逐项填写实盘数量
6. 提交后系统自动计算差异 = 实盘 - 系统
7. 只展示有差异的行
8. 差异大的行（差异率>5%）标记"需复盘"
9. 仓库主管审核差异 → 确认后系统自动调整库存
10. 盘点期间被盘库位冻结，禁止出入库

**边界场景：**

- 盘点期间有人尝试出入库 → 提示"该库位正在盘点"
- 盘点差异率超过20% → 强制要求复盘

---

### US-INV-008 库存预警

**作为** 仓库主管  
**我希望** 设置库存预警规则  
**以便** 及时发现低库存或积压

**验收标准：**

1. 可配置预警规则：低库存、积压、库龄超期
2. 低库存阈值可按品类分别设置（如成品≤10件、面料≤可用7天）
3. 系统每小时扫描一次，生成预警记录
4. 预警在首页仪表板展示
5. 预警可标记"已确认"或"已解决"

---

### US-INV-009 库存操作流水查询

**作为** 仓库主管/系统管理员  
**我希望** 查询任意SKU/物料的库存变更历史  
**以便** 追溯库存异常

**验收标准：**

1. 按SKU/物料+仓库+时间范围查询操作流水
2. 每条流水展示：操作类型、操作数量、变更前数量、变更后数量、来源单据、操作人、时间
3. 支持按操作类型筛选
4. 流水不可修改和删除

---

## 三、业务规则

| 编号 | 规则 |
|------|------|
| BR-INV-001 | 可用库存不可为负，扣减前必须校验 |
| BR-INV-002 | 所有库存变更必须写操作流水（记录before/after） |
| BR-INV-003 | 禁止直接UPDATE库存数量字段 |
| BR-INV-004 | 销售出库从锁定库存扣，领料出库从可用库存扣 |
| BR-INV-005 | 盘点期间库位冻结 |
| BR-INV-006 | 预留超期自动释放（默认7天，可配置） |
| BR-INV-007 | 出库批次推荐FIFO，同款同色同批优先 |
| BR-INV-008 | 调拨不改变批次号 |
