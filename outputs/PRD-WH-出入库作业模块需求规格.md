# PRD-WH 出入库作业模块需求规格

> 模块编号：WH  
> 优先级：P0（第二/三期交付）  
> 前置依赖：基础数据、库存管理模块（库存操作、库位）、采购管理（ASN）、订单管理（出库需求）

---

## 一、模块概述

出入库作业模块是仓库人员的日常操作界面，将库存管理模块定义的数据规则转化为具体的操作步骤：扫码收货、质检录入、库位上架、波次拣货、复核发货。

---

## 二、用户故事

### US-WH-001 收货作业

**作为** 仓管员  
**我希望** 扫码完成收货操作  
**以便** 快速准确记录到货物料

**验收标准：**

1. 扫ASN单号条码，加载到货通知单信息
2. 逐行核对物料：扫物料条码确认，填写实收数量
3. 面料额外填写实收卷数
4. 实收与预计有差异时需选择差异原因（多收/少收/破损）
5. 支持部分收货（一次到一部分）
6. 每次收货系统自动生成批次号
7. 收货确认后自动触发库存变更（在途→质检）

---

### US-WH-002 来料检验作业

**作为** 质检员  
**我希望** 在PDA/PC上录入检验结果  
**以便** 确认物料质量是否达标

**验收标准：**

1. 待检验列表展示所有到货未检的物料行
2. 选择一行进入检验页面，逐项填写检验结果
3. 面料检验页面预置：色差、克重、门幅、缩水率、色牢度检验项
4. 每项填写标准值和实测值，系统自动判定合格/不合格
5. 检验结论：合格/不合格/让步接收
6. 检验合格 → 自动触发库存变更（质检→可用）
7. 检验不合格 → 自动生成退货出库单

---

### US-WH-003 上架作业

**作为** 仓管员  
**我希望** 系统推荐上架库位，扫码确认上架  
**以便** 货物快速入位，方便后续查找

**验收标准：**

1. 系统推荐库位（优先同物料已有库存的库位）
2. 仓管员可接受推荐或手动指定库位
3. 扫库位条码 → 扫物料条码 → 确认上架
4. 上架后更新库存记录的库位信息
5. 支持同一收货行分多个库位上架（一个库位放不下时）

---

### US-WH-004 波次管理

**作为** 仓库主管  
**我希望** 将多个出库需求合并为波次  
**以便** 批量拣货提高效率

**验收标准：**

1. 可手动创建波次，选择待出库的订单
2. 可按策略自动生成波次：按时间段、按订单量、按物流商
3. 波次释放时系统自动分配库位和批次
4. 释放后生成拣货单
5. 波次状态：已创建→已释放→拣货中→分拣中→已完成
6. 可查看波次包含的订单列表和拣货进度

---

### US-WH-005 拣货作业

**作为** 仓管员  
**我希望** 按优化路径的拣货单逐行拣货  
**以便** 高效完成拣货任务

**验收标准：**

1. 拣货单行按路径优化排序（拣货位优先、S型走位、同款同色尺码相邻）
2. 逐行拣货：扫库位码→扫SKU码→确认数量
3. 实拣数量不足时填写缺货数量和原因
4. 支持逐单拣和汇总拣两种模式
5. 汇总拣完成后需进入分拣步骤
6. 拣货单可打印（纸质备用）

---

### US-WH-006 分拣作业

**作为** 仓管员  
**我希望** 将汇总拣出的货物按订单分配  
**以便** 每个订单的货物集中到一起

**验收标准：**

1. 汇总拣货完成后进入分拣环节
2. 逐件扫码，系统自动识别该分配给哪个订单
3. 分拣区设置分拣格（每格对应一个订单）
4. 系统提示当前SKU应放入哪个格子
5. 分拣完成后每个订单的货物集中在对应格子中

---

### US-WH-007 复核作业

**作为** 仓管员  
**我希望** 逐单扫码复核出库货物  
**以便** 确保出库的款式、颜色、尺码、数量正确

**验收标准：**

1. 逐单复核：扫出库单号进入复核模式
2. 逐件扫码核对SKU，系统自动比对订单需求
3. 服装行业复核重点提示：款式正确性、颜色不混淆、尺码不错配
4. 复核通过 → 进入打包环节
5. 复核不通过 → 退回拣货区重新拣
6. 复核结果记录（通过/不通过、不通过原因）

---

### US-WH-008 打包与发货

**作为** 仓管员  
**我希望** 打包后确认发货  
**以便** 货物交接给物流

**验收标准：**

1. 选择包装方式：折叠装箱/挂装出货/混合
2. 贴物流面单（打印或手工粘贴）
3. 放装箱单（随货同行）
4. 称重记录
5. 发货确认 → 扣减库存（锁定库存减少）、更新销售订单状态
6. 填写物流公司和物流单号

---

### US-WH-009 领料出库

**作为** 仓管员  
**我希望** 按生产订单的BOM需求领料出库  
**以便** 车间拿到面料和辅料开始生产

**验收标准：**

1. 选择生产订单，系统根据BOM+预留自动生成领料出库单
2. 面料行指定批次（同款同色同批优先）
3. 面料领出支持两种模式：整卷领出/裁剪领出
4. 整卷领出时出库数量为整卷米数
5. 辅料按预留数量出库
6. 确认出库后可用库存扣减

---

### US-WH-010 退料管理

**作为** 仓管员  
**我希望** 管理车间的退料入库  
**以便** 未用完的原料回到仓库

**验收标准：**

1. 车间发起退料申请，填写退回物料和数量
2. 退回批次号必须与领出批次号一致
3. 面料退料需测量实际退回米数
4. 仓库确认收货 → 可用库存增加
5. 更新生产订单的领料记录

---

### US-WH-011 条码与打印

**作为** 仓管员  
**我希望** 打印SKU标签和各类单据  
**以便** 贴标和随货同行

**验收标准：**

1. SKU标签打印：选择SKU，批量打印标签（含SKU编码、款名、颜色、尺码、条码）
2. 箱唛标签打印：填写箱号和内容物，打印箱唛
3. 入库单打印：A4纸，含物料明细、库位、数量
4. 拣货单打印：A4纸，含库位→SKU→数量的有序列表
5. 出库单打印：A4纸，含订单信息、SKU明细
6. 装箱单打印：随货同行，含收货方、SKU明细
7. 打印模板可配置（后续扩展）

## 三、业务规则

| 编号 | 规则 |
|------|------|
| BR-WH-001 | 不同次到货必须分配不同批次号 |
| BR-WH-002 | 检验合格才可上架，不合格走退货 |
| BR-WH-003 | 盘点冻结的库位禁止出入库操作 |
| BR-WH-004 | 拣货路径按S型走位优化，同款同色尺码相邻 |
| BR-WH-005 | 发货确认才扣库存，出库单创建不扣 |
| BR-WH-006 | 同款同色面料出库优先同批次 |
| BR-WH-007 | 退料批次号必须与领出批次一致 |
